## 📚 Sellio Metrics Wiki

This wiki is a deeper, code-level guide to the Sellio Metrics project. It complements the root `README.md`, `ARCHITECTURE.md`, and the backend/frontend READMEs by organizing the information into focused, navigable pages.

- **System overview**: high-level architecture, capabilities, and data flow — see [`system-overview.md`](system-overview.md)
- **Backend architecture**: Workers/TypeScript backend, modules, caching, error handling, and GitHub integration — see [`backend-architecture.md`](backend-architecture.md)
- **Frontend architecture**: Flutter Web layers, state management, routing, and design system — see [`frontend-architecture.md`](frontend-architecture.md)
- **Domain & metrics model**: how PR metrics, bottlenecks, leaderboards, and KPIs are computed — see [`domain-and-metrics.md`](domain-and-metrics.md)
- **Development & operations**: local dev, testing, CI/CD, deployment, and environment configuration — see [`development-and-ops.md`](development-and-ops.md)

For a diagram-first perspective, start with `ARCHITECTURE.md` in the repo root, then drill into the relevant wiki pages from here.
